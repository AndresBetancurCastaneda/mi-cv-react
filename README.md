# Descripción general

El proyecto consiste en la creación de una **hoja de vida modular (CV)** desarrollada con **React**, donde cada sección (perfil, experiencia, educación, habilidades, etc.) se implementa como un componente funcional.  
Además, se aplican buenas prácticas en el manejo de dependencias, estructura de carpetas y control de versiones con **Git y GitHub**.  

Estas actividades permiten reforzar el aprendizaje sobre:
- Instalación y configuración de entornos con Node.js y Vite.  
- Creación y modularización de componentes en React.  
- Manejo y actualización de dependencias con npm.  
- Uso de Git y GitHub para el control de versiones.
